Structure de Dossier
####################

Par exemple, si vous voulez traduire Croogo dans
Bengali (dont le code de langue ISO 639-2 est ben), la structure de dossier
sera:

  - app/Locale/ben/
    - LC_MESSAGES/
      - default.po
